Routes

Healthcheck:

  • GET /healthcheck : returns API health status

Version:

  • GET /version : returns API version

Analysis:

  • GET /analysis/:id : gets an analysis given its id (RID)
  • POST /analysis : creates a new analysis

SecurityTest:

  • POST /securitytest : creates a new security test

Repository:

  • POST /repository : creates a new repository
