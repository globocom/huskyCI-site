## Requirements

### Node

If you don't have Node installed in your environment, follow the instructions [here](https://gist.github.com/d2s/372b5943bce17b964a79) for your correspondent OS.

### Yarn

```sh
npm install -g yarn
```

## Running

```sh
npm start
```
